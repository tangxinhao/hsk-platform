# 题目类型使用指南

## 📖 概述

HSK学习系统已将题目类型简化为7种主要类型，本指南说明如何在后台添加题目和在前台使用题目。

## 🎨 7种题目类型详解

### 1. 听力题 (listening)

**适用场景**:
- 所有需要听音频的题目
- 包括：图片判断、图片选择、对话配图、文字选择、长对话理解等

**添加题目时需要**:
- ✅ 题目内容：问题描述
- ✅ 选项：A、B、C、D等选项
- ✅ 正确答案：如 "A" 或 "B"
- ✅ 音频文件：上传或填写音频URL
- 📋 音频时长：填写音频秒数
- 📋 音频分组：多题共享音频时使用

**示例**:
```
题目类型: 听力题
题目内容: 请听音频，选择正确的图片
选项: ["图片A", "图片B", "图片C", "图片D"]
答案: B
音频URL: http://localhost:8000/media/audio/H11556.mp3
音频时长: 20
```

---

### 2. 阅读题 (reading)

**适用场景**:
- 所有需要阅读文字的题目
- 包括：图片词语匹配、句子图片匹配、对话匹配、选词填空、阅读理解等

**添加题目时需要**:
- ✅ 题目内容：问题或文章
- ✅ 选项：A、B、C、D等选项
- ✅ 正确答案：如 "C"
- 📋 文章内容：如果是阅读理解，填写passage字段
- 📋 文章标题：可选

**示例**:
```
题目类型: 阅读题
题目内容: 下列词语与图片对应正确的是？
选项: ["苹果", "香蕉", "橙子", "西瓜"]
答案: A
图片URL: http://localhost:8000/media/images/apple.jpg
```

---

### 3. 书写题 (writing)

**适用场景**:
- 需要学生书写或组织文字的题目
- 包括：组句、汉字填空、作文、文章缩写、看图描述等

**添加题目时需要**:
- ✅ 题目内容：题目要求
- ✅ 正确答案：标准答案或参考答案
- 📋 图片：如果是看图写作
- 📋 说明：作文要求、字数限制等

**示例**:
```
题目类型: 书写题
题目内容: 请将下列词语组成一个完整的句子：我/学校/去/今天
答案: 今天我去学校。
```

---

### 4. 填空题 (fill)

**适用场景**:
- 需要填写答案的题目
- 包括：普通填空、完形填空、词语填空等

**添加题目时需要**:
- ✅ 题目内容：带空格的句子或文章
- ✅ 正确答案：空格应填的内容
- 📋 选项：如果是选词填空，提供选项

**示例**:
```
题目类型: 填空题
题目内容: 我今天___去学校。（没有/不）
答案: 没有
选项: ["没有", "不"]
```

---

### 5. 单选题 (single)

**适用场景**:
- 传统单项选择题
- 包括：一般单选、看图选择、匹配题、排序题等

**添加题目时需要**:
- ✅ 题目内容：问题
- ✅ 选项：A、B、C、D等选项
- ✅ 正确答案：一个选项，如 "D"

**示例**:
```
题目类型: 单选题
题目内容: "再见"的拼音是？
选项: ["zaijian", "zàijiàn", "zai jian", "zài jiàn"]
答案: D
```

---

### 6. 多选题 (multiple)

**适用场景**:
- 可以选择多个正确答案的题目

**添加题目时需要**:
- ✅ 题目内容：问题
- ✅ 选项：A、B、C、D等选项
- ✅ 正确答案：多个选项，如 "A,C,D" 或 ["A","C","D"]

**示例**:
```
题目类型: 多选题
题目内容: 以下哪些是水果？
选项: ["苹果", "西红柿", "香蕉", "黄瓜"]
答案: A,C
```

---

### 7. 判断题 (judge)

**适用场景**:
- 判断对错的题目
- 二选一的题目

**添加题目时需要**:
- ✅ 题目内容：陈述句
- ✅ 正确答案："对" 或 "错" / "正确" 或 "错误" / "A" 或 "B"

**示例**:
```
题目类型: 判断题
题目内容: "你好"是中文的问候语。
答案: 对
```

## 🎯 后台添加题目步骤

### 步骤1: 选择题目类型
1. 登录后台管理系统
2. 进入"题目管理"页面
3. 点击"新增题目"
4. 在"题目类型"下拉框中选择7种类型之一
5. 查看题型说明，了解该类型的用途

### 步骤2: 填写基本信息
- **题目内容** (必填): 题目的问题或描述
- **HSK等级** (必填): 选择1-6级
- **难度系数** (必填): 选择1-5星

### 步骤3: 填写选项和答案
- **选项**: 点击"添加选项"按钮，逐个添加选项
- **正确答案** (必填): 填写正确答案（如"A"、"B"等）

### 步骤4: 添加多媒体内容（可选）
- **听力题**: 填写音频URL和音频时长
- **阅读题**: 如果是阅读理解，填写文章内容
- **书写题**: 如果是看图写作，填写图片URL

### 步骤5: 保存题目
- 检查所有信息是否正确
- 点击"保存"按钮
- 保存成功后，题目会出现在题目列表中

## 🔍 前台使用题目

### 小程序端筛选
1. 打开题库练习或题目列表页面
2. 点击"题目类型"筛选器
3. 选择想要练习的题型：
   - 全部类型
   - 听力题
   - 阅读题
   - 书写题
   - 填空题
   - 单选题

### 按难度练习
1. 选择HSK等级（1-6级）
2. 选择题目类型
3. 开始练习

### 查看题目详情
- 题目卡片显示题目类型标签
- 点击题目进入详情页
- 如果是听力题，可以播放音频
- 如果有图片，可以查看大图

## 💡 最佳实践

### 添加题目时
1. **选择合适的类型**: 根据题目特点选择最匹配的主类型
2. **填写完整信息**: 尽量填写所有相关字段
3. **使用标签**: 如需细分，可在题目内容或说明中标注
4. **测试音频**: 添加听力题后，测试音频是否能正常播放
5. **检查答案**: 确保答案格式正确（如单选用"A"，多选用"A,C"）

### 题目内容编写
1. **清晰简洁**: 题目描述要清晰明了
2. **符合等级**: 题目难度要符合选择的HSK等级
3. **选项均衡**: 单选题的干扰项要有一定迷惑性
4. **答案准确**: 确保答案唯一且正确

### 多媒体文件
1. **音频格式**: 推荐使用MP3格式
2. **图片格式**: 推荐使用JPG或PNG格式
3. **文件大小**: 音频不超过5MB，图片不超过2MB
4. **文件路径**: 确保文件已上传到服务器的正确目录

## ❓ 常见问题

### Q1: 我的题目既有音频又有阅读，应该选哪个类型？
**A**: 选择"听力题"，因为听力是主要技能。可以在题目内容中包含文字材料。

### Q2: 完形填空应该选哪个类型？
**A**: 选择"填空题"。如果是阅读理解式的完形填空，也可以选择"阅读题"。

### Q3: 看图选择应该选哪个类型？
**A**: 选择"单选题"，并上传图片。

### Q4: 原来的细分类型还能用吗？
**A**: 系统已将所有细分类型统一为7种主要类型。原始类型信息已保存，但不再显示。

### Q5: 如何区分同类型中的不同题目？
**A**: 
- 使用题目内容描述清楚题型特点
- 使用HSK结构字段（section_type, part_number）
- 使用标签（tags）字段标注细分信息

### Q6: 多选题的答案格式是什么？
**A**: 可以使用 "A,C,D" 或 ["A","C","D"] 格式，系统都能识别。
