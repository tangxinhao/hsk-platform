# 验证并创建证书目录

## 步骤 1：验证目录是否已创建

在 FinalShell 终端中执行：

```bash
ls -la /home/hsk/ssl/
```

如果看到 `tangxinhao.online` 文件夹，说明创建成功了。

如果提示 "No such file or directory"，说明目录还没创建，继续下一步。

---

## 步骤 2：创建目录（如果不存在）

在 FinalShell 终端中执行：

```bash
cd /home/hsk
mkdir -p ssl/tangxinhao.online
ls -la ssl/
```

应该看到 `tangxinhao.online` 文件夹。

---

## 步骤 3：刷新 FinalShell 文件浏览器

**方法一：点击刷新按钮**
- 在 FinalShell 文件浏览器顶部，找到**刷新**按钮（通常是圆形箭头图标）
- 点击刷新，应该就能看到 `ssl` 文件夹了

**方法二：重新进入目录**
- 在文件浏览器左侧目录树中，点击其他文件夹，然后再点击 `/home/hsk`
- 或者直接点击路径栏，按回车

**方法三：在文件浏览器中直接创建**
- 在 FinalShell 文件浏览器中，右键点击 `/home/hsk` 目录
- 选择 **新建文件夹**
- 输入 `ssl`，回车
- 进入 `ssl` 文件夹，再右键 → **新建文件夹** → 输入 `tangxinhao.online`

---

## 步骤 4：验证最终结果

在终端执行：

```bash
ls -la /home/hsk/ssl/tangxinhao.online/
```

应该显示空目录（因为还没上传证书文件）。

---

## 如果还是看不到

尝试在文件浏览器中直接创建：

1. 在 FinalShell 文件浏览器左侧，找到 `/home/hsk`
2. 右键点击 `/home/hsk` → **新建文件夹** → 输入 `ssl`
3. 双击进入 `ssl` 文件夹
4. 右键空白处 → **新建文件夹** → 输入 `tangxinhao.online`

这样应该就能看到了！
