## 注意事项
1. 所有图标应保持统一风格
2. 导航图标尺寸建议为 81px × 81px
3. 功能图标尺寸建议为 64px × 64px
4. 所有图标应导出为 PNG 格式，支持透明背景
5. 选中状态的导航图标应使用主题色 #409EFF
