## 注意事项
1. 所有图片应保持适当的分辨率和文件大小，避免过大导致加载缓慢
2. 轮播图建议尺寸为 750px × 300px
3. 高校图片建议尺寸为 300px × 180px
4. 文化内容图片建议尺寸为 350px × 200px
5. 校徽图片应为正方形，建议尺寸为 100px × 100px
